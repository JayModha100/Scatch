# Scatch — Backend-Focused E-Commerce Platform

Scatch is a backend-focused e-commerce application built to learn real-world backend engineering concepts beyond tutorial-level CRUD development. The project emphasizes authentication systems, modular backend architecture, request lifecycle understanding, middleware-based authorization, database modeling, and production-oriented backend practices using Node.js, Express.js, MongoDB, and Mongoose.

## Features

* User Registration & Login
* JWT-based Authentication
* Secure Password Hashing using bcrypt
* Cookie-based Session Handling
* Protected Routes using Middleware
* MongoDB Integration with Mongoose
* Admin/Owner Route Structure
* Product Schema & User Schema Design
* Flash Messaging System
* Express Session Management
* Modular Backend Architecture
* Environment Variable Configuration
* Retry-Based MongoDB Connection Logic
* Infrastructure Startup Sequencing

## Backend Concepts Implemented

### Authentication & Security

* JWT token generation and verification
* Cookie-based authentication flow
* Password hashing and comparison
* Protected route middleware
* Session and flash middleware integration

### Backend Architecture

* Separation of routes, controllers, models, middleware, config, and utilities
* Modular Express.js application structure
* Request lifecycle management
* Startup dependency sequencing
* Infrastructure configuration abstraction

### Database Engineering

* MongoDB schema design using Mongoose
* User and product data modeling
* Basic database relationship preparation
* Async database interaction handling
* Connection retry strategy implementation

### Engineering-Oriented Practices

* Environment variable usage with dotenv
* Failure-aware MongoDB connection setup
* Modular code organization
* Reusable authentication utilities
* Async backend workflow handling
